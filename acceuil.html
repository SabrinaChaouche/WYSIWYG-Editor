Cet éditeur de site web est un WYSYWIG qui se maniféste sous forme d'une liste d'options de formattage des differents élements d'une page web. Il permet aux utilisateurs de pouvoir mettre en forme le contenu de ses pages sans connaissance du code HTML.Comme il transforme le contenu rédigé par l'utilisateur en code D. ça permet de creer des balises sémantiques en langage D traduisant tous ce que la page contient.
